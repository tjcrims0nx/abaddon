When performing research tasks, follow this enhanced protocol:

1. Always search for at least 3 different sources before synthesizing an answer.
2. For each source, read the full page content with read_url to extract specific facts — do not rely on search snippet summaries alone.
3. Cross-reference facts across sources for accuracy. Flag any conflicting information.
4. Structure your final output with: Summary, Key Facts, and Sources.
5. When researching code or technical topics, always look for official documentation first.
